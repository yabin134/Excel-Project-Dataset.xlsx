# Bike Sales Dashboard - Excel Project

## Overview

This project analyzes customer bike purchase behavior using Microsoft Excel. The dataset was cleaned, transformed, and analyzed to identify trends based on demographics, income, age, occupation, education, and commute distance.

## Objectives

* Clean and prepare raw sales data
* Analyze customer purchasing patterns
* Create Pivot Tables and Pivot Charts
* Build an interactive dashboard for data-driven decision-making

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Data Cleaning
* Data Visualization

## Key Insights

* Middle-aged customers (31–54 years) showed the highest bike purchase rates.
* Higher-income customers were more likely to purchase bikes.
* Commute distance influenced purchasing decisions.
* Customer demographics such as gender, occupation, and education impacted sales trends.

## Dashboard Features

* Interactive slicers for:

  * Marital Status
  * Occupation
  * Education
  * Gender
* Average Income Analysis
* Commute Distance Analysis
* Age Group Analysis
* Purchase Behavior Comparison

## Project Outcome

Developed an interactive Excel dashboard that transforms raw customer data into actionable business insights, demonstrating skills in data analysis, visualization, and reporting.
